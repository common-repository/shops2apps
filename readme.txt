== Installation ==
1. Go to WordPress Control Panel
2. Click "Plugins", then "Add New"
3. Type in " Shops2apps" and click "Search Plugins"
4. Click the "Install" button
5. Click the "Activate Plugin" link
6. Move to the "Shops2apps" section in the menu and start customizing your app
7. Done!
